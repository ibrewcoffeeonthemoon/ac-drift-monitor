# AC Drift Monitor

Visualize car telemetry when drifting.

## Install

1. install Assetto Corsa Content Manager
2. clone this repo into `assettocorsa/apps/python/`
3. you should have a folder named `assettocorsa/apps/python/ac-drift-monitor/`
4. enable the app under `settings / ASSETTO CORSA / PYTHON APPS`

## Load

1. enter game, move mouse cursor to right corner and click app drawer icon
2. under `Your apps`, find `AC Drift Monitor` and click to open app window
3. move the app to appropriate location, click the pin icon to lock it in place
